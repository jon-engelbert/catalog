=======

To run this project

1. Move this catalog directory inside the vagrant environment

2a. run database_setup.py to reset the database
2b. run lotsofmenus2.py to reset the database

To install the required python libraries...
3. sudo pip install flask-wtf
4. sudo pip install oauth2client

5. run finalproject.py and navigate to 127.0.0.1:8000 in your browser

For adding OAuth handling

pip install --upgrade oauth2client simplekv flask_kvsession
